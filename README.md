# 요구사항 분석

## 개요

- 회원을 등록할 수 있어야 한다.
- 휴가 종류를 등록할 수 있어야 한다.
- 회원에 의해 휴가 신청 및 결재(수락/거부)가 가능해야 한다.

## 상세

### 회원 기능

- 회원 등록
- 회원 조회
- 결재자 변경

### 휴가 기능

- 휴가 등록
- 휴가 조회
- 휴가 수정

### 결재 기능

- 휴가 결재 (수락 / 거절)
- 결재 내역 조회

### 신청 기능

- 휴가 신청
- 신청 내역 조회
- 신청 취소

### 부여 기능

- 휴가 부여
- 남은 휴가 확인

### 푸시 기능

- 휴가 신청, 취소 시 결재자에게 푸시
- 휴가 수락, 반려 시 신청자에게 푸시
